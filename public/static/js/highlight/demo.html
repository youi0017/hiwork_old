3. 在全局注册组件

在 根目录的 main.js 文件中加入如下内容：

// 导入自定义的全局组件 components
		import myhHeader from './components/myh/myh-header.vue'
		import myhFooter from './components/myh/myh-footer.vue'
		
		
		// 注册为全局组件
		Vue.component("myh-header",myhHeader)
		Vue.component("myh-footer",myhFooter)
